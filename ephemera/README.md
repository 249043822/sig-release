# Ephemera

This directory was intended to be short lived. It is populated with files
that were migrated of out [kubernetes/community](https://git.k8s.io/community)
in an effort to centralize all pertinent information for SIG Release or the
kubernetes/kubernetes release process. They were dropped in this directory
with the intent that they would be pruned or migrated to more suitable
locations over time. One day, maybe we will accomplish this goal.

If you came here looking for a description of release team roles, please see
[the release-team README](/release-team/README.md)
